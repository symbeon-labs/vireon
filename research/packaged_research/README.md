# VIREON Research Package

Este pacote contém todos os dados de pesquisa, análises e insights do projeto VIREON.

## Estrutura do Pacote

- `consolidated_report.json`: Relatório consolidado com todos os insights e análises
- `INDEX.md`: Índice estruturado de todos os arquivos
- Diretórios originais preservados com seus arquivos

## Como Usar

1. Navegue pelo `INDEX.md` para encontrar arquivos específicos
2. Consulte `consolidated_report.json` para ver todos os insights consolidados
3. Explore os diretórios originais para acessar arquivos específicos
