# Shared Components

Componentes compartilhados entre VIREON-CORE e VIREON-NEURAL.

## Estrutura

1. Interfaces
   - Definições de tipos
   - Protocolos de comunicação
   - Bridges de integração

2. Configuração
   - Parâmetros compartilhados
   - Validação de configuração
   - Perfis de uso

3. Utilitários
   - Funções helpers
   - Métricas compartilhadas
   - Validadores

## Uso

Estes componentes fornecem a base para integração segura entre os módulos práticos e experimentais.
